# 移动端适配问题

在移动端打开时，因为搜索框的问题，会导致页面可以左右滑动，可以通过设置meta标签值解决问题，就跟我们写移动端项目添加meta一样

module.exports = {
  head: [
    ['meta', { name: 'viewport', content: 'width=device-width,initial-scale=1,user-scalable=no' }]
  ]
}

# 安装/使用

vuepress官网

  • 新建项目

  • 快速初始化项目

yarn init -y
或者：npm init -y
1
2
  • 全局安装vuepress
yarn global add vuepress 
或者：npm install -g vuepress
1
2
  • 在项目中新建文件夹docs
mkdir docs
或者：直接在项目文件夹下右键--新建文件夹
1
2
  • 配置package.json文件
{
  "scripts": {
    "docs:dev": "vuepress dev docs", //本地开发
    "docs:build": "vuepress build docs" //打包
  }
}

  • 运行项目
yarn docs:dev
或者：npm run docs:dev

2

Tips：项目默认是8080端口，如果端口被占，可以通过.vuepress/config.js中port字段值进行修改

module.exports = {
    port: 8081
}
